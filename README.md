Este proyecto es una página web dedicada al creador de contenido **RickyEdit**, construida utilizando **Astro** y **Tailwind CSS**.

<a href="https://rickyedit.pages.dev/">
  <img src="https://img.shields.io/badge/☁_VIEW_LIVE_SITE-FF7043?style=for-the-badge&logoColor=white" alt="Live Demo" />
</a>


## ✨ Características

* **Reproductor de Vídeos:** La web incluye una sección especial donde se pueden ver las últimas subidas del canal de RickyEdit de forma integrada.
* **Interfaz Adaptable:** El diseño está pensado para verse bien tanto en ordenadores como en dispositivos móviles, con bordes redondeados y sombras modernas.

## 🛠️ Detalles del Proyecto

* **Fuente de Contenido:** Los vídeos se obtienen directamente de la lista de reproducción de "Subidas" del canal oficial.
* **Tecnologías:** Astro para la estructura y Tailwind CSS para todos los estilos visuales.
